Project

Potensiometer

Koblingsmåte:

Wiperen til potensiometeren blir koblet til A1 på arduinoen.
Pluss eller 5V på arduinoen blir koblet til venstre siden av potensiometeren, og minus til ground

Anoden på dioden kobles til input -5
Katoden kobles til 220 ohm og den andre beinen til motstanden som er koblet til jord

Kode:
const int pot = A1; // arduino pin A1 or 15
const int led = 5;
int potIn;
int value;
int dutyC;
void setup()
{
pinMode(pot, INPUT);
Serial.begin(9600);
}
void loop()
{
potIn = analogRead(pot);
value = map(potIn, 0, 1023, 0, 255);
dutyC = map(potIn, 0, 1023, 0, 100);
analogWrite(led, value);
Serial.print("Duty Cycle: ");
Serial.print(dutyC);
Serial.println("%");
delay(100);
}
